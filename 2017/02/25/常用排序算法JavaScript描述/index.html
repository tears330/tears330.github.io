好久不更新文章了，说白了还是懒啊-。-最近在准备面试，正好重新温习一下常见的排序算法用JavaScript实现一遍。

冒泡排序

所谓冒泡排序就是顾名思义，做arr.length次数的循环，每次循环中都比较两两相邻的元素，让较大的元素浮上来，循环结束即排序完成。

其实复杂度为n^2的lowB算法还有选择排序、插入排序，但是实现的方式难度大同小异，这里选择一个比较有代表性的实现。

快速排序

非常经典的排序算法，引用了递归分治的思想。

  1. 从数组中随机选出一个值a作为参照，将比大于a的值放到右边的数组，将小于a的值放在左面的数组。

  2. 分别对左面的数组和右面的数组再次进行步骤一的操作。直到当数组仅剩一个值或没有值时结束操作。
