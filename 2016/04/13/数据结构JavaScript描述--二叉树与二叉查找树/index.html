数据结构JavaScript描述--二叉树和二叉查找树

发表于 2016-04-13 | 分类于 JavaScript |

树是一种非线性的数据结构，以分层的方式存储数据。树被用来存储具有层级关系的数据，例如文件系统中的文件。

树的定义

树是一组以边连接的节点组成。下面的图展示了关于树的常用术语：

一个树最上面的节点称作根节点；如果一个节点下面连接多个节点，那么该节点成为父节点，它下面的节点叫做子节点；没有任何子节点的节点称作叶子节点。

我们所讨论的二叉树是一种特殊的树，它的子节点个数不超过两个。这使他具有一些特殊的计算性质，让他们在一些操作上异常高效。

二叉树与二叉查找树

二叉树每个节点的子节点不允许超过两个。通过将子节点的个数限定为2，可以写出高效的程序在树种插入，查找，删除数据。

而二叉查找树是一种特殊的二叉树，相对较小的值保存在子节点中，较大的值保存在右节点中。这一特性使得查找的效率非常高。

我们需要遍历BST，将节点插入适当位置。算法如下：

  1. 设置根节点为当前节点。

  2. 如果待插入的节点保存数据小于当前节点，则设新的当前节点为原节点的左节点；不过不是进入步骤4.

  3. 如果当前节点的左节点为null，就讲新的节点插入这个位置，退出循环；如果不是，则再次循环。

  4. 设新的当前节点为原节点右节点。

  5. 如果当前节点的右节点为null，就将新的节点插入这个位置，退出循环；如果不是，则再次循环。
