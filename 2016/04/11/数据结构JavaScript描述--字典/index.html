发表于 2016-04-11 | 分类于 JavaScript |

字典是一种以键-值对形式储存数据的数据结构。就像电话簿里名字和号码的关系一样，名字是你用来查找的东西就是键，而号码是查找得到的结果就是值。

JavaScript中Object类就是以字典的形式设计的。我们可以使用Object类本身的特性，实现一个Dictionary类。
