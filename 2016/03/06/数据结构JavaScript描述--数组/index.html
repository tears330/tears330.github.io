数据结构JavaScript描述--数组

发表于 2016-03-06 | 分类于 JavaScript |

数组是一种最最常见的数据结构，通常编程语言的将数组作为内建类型，效率很高，可以满足不同需求的数据存储。

JavaScript中对数组定义

数组的标准定义是：一个储存元素的线性集合，元素可以通过索引来任意存取，索引通常是数字，用来计算元素之间储存位置的偏移量。

在JavaScript中数组是一种特殊的对象，用来表示偏移量的索引是该对象的属性，索引可能是整数。然而这些数字索引在内部被转换为字符串类型，这是因为JavaScript对象中的属性名必须是字符创。数组在JavaScript中只是一种特殊的对象，所以效率上不如其他的语言中数组效率高。

使用JavaScript数组

split()由字符串生成数组

调用字符串对象的split()方法，接受字符串类型的参数。并将调用该方法的字符串对象，按照传递的参数分割成数组。

数组的复制

在JavaScript中只有对基本类型的值直接复制，传递的内容才是值。而对引用类型式，传递的内容是该引用对象的引用。

存取函数

JavaScript提供了一组用来访问数组元素的函数，叫做存取函数，这些函数返回目标数组的某种变体。

查找元素

indexOf()是最常用的存取函数之一，用来查找传递的参数在目标数组中是否存在。若果目标数组包含该参数，就返回该元素在数组中的索引；如果不包含，就返回-1.

可以看到当数组包含多个相同元素是，indexOf()总是返回第一个与参数相同的元素索引。另外还有一个与之相似的函数lastIndexOf()，该函数返回相同元素的最后一个元素索引，如果不存在则返回-1。

数组的字符串表示

由两个方法可以将数字转化为字符串：join()和toString()。这两个方法都返回一个包含数组所有元素的字符串，各元素直接用逗号分隔开。

由已有数组创建新数组

concat()和splice()方法允许通过已有数组创建新数组。concat()方法可以合并多个数组创建一个新数组，splice()方法截取一个数组的自己创建一个新数组。

concat()方法

concat()方法接受一个数组类型的参数并将该参数中所有元素连接到调用concat()数组的后面。

注意如果合并的数组中包含引用类型，合并后新数组中的引用类型依然是对原数组中引用类型的引用！

splice()方法

splice()方法从现有数组里截取一个新数组。该方法第一个参数是截取的起始索引，第二个是截取长度。

可变函数

JavaScript拥有一组可变函数，通过他们可以不必引用数组中的某个元素，就能改变数组内容。

为数组排序

reverse()方法

该方法将数组中元素的顺序进行翻转。

sort()方法

当数组中元素为字符串式，使用sort()可以将数组按首字母顺序排序。

但是当元素为数字类型时，sort()的排序方案就开始混乱：

sort()方法是按照字典顺序对元素进行排序，因此它假定数组里的所有元素都是字符串。我们可以为sort()方法传递一个比较大小的函数，排序时，sort()方法会根据该函数比较数组中两个元素大小从而决定顺序。

迭代器方法

迭代器方法是对数组中的每个元素应用一个函数，可以返回一个值、一组值或者一个新数组。

不生成新数组的迭代器方法

这类的方法不产生任何新的数组，相反，他们要么对于数组中的每个元素执行某种操作，要么返回一个值。

forEach()方法

该方法接受一个函数作为参数，对数组中的每个元素都使用该函数。

every()方法

该方法接受一个返回值为Boolean类型的函数，对数组中的每个元素使用该函数。如果对于所有的元素，该函数的函数返回为true则该方法返回true否则返回false。

some()方法

some()方法同样接受一个返回值为Boolean类型的函数，只要有一个元素使得该函数返回true，该方法就返回true。

reduce()方法

reduce()接受一个函数作为参数，返回一个值。该方法会从一个累加值开始，不断的对累加值和数组中的后续元素调用该函数，直到数组中的最后一个元素，最后返回这个累加值。

生成新数组的迭代器方法

有两个迭代器方法可以产生新数组：map()和filter()。

map()方法

map()和forEach()有些类似，对数组中的每一个元素使用参数中函数。区别是map()方法会返回一个由原有元素应用参数函数后返回值组成的数组。

filter()方法

filter()方法类似于every()方法，传入一个返回值为Boolean类型的函数。该方法会返回一个新数组，每个元素都是对原有数组中应用函数且返回值为true的元素。
