数据结构JavaScript描述--栈

发表于 2016-03-08 | 分类于 JavaScript |

栈是一种特殊的列表，栈内的元素只能通过列表的一段访问，这一端称为栈顶。所以，栈被称为一种后进先出的数据结构（LIFO,last-in-first-out）。

由于栈后进先出的特点，所以任何不在栈顶的元素都无法访问。为了得到栈底元素必须先拿掉上面的元素。

栈的ADT

属性和方法 含义
dataStore 初始化数组储存空间

pop 将一个元素弹出栈顶
peek 预览栈顶元素的值
clear 清空栈内所有元素
length 返回栈中元素数量

数制间相互转换

利用栈可以将一个数字从一种数制转换成另一种数制。

讲一个数字n转换为以b为基数的数字，转换算法如下：

  1. 最高位为n%b,将此为压入栈
  2. 使用n/b代替n
  3. 重复步骤1和2，直到n等于0且没有余数
  4. 将栈内所有元素弹出，依次将这些元素排列，就是转换后的数字

判断回文

回文是指一个字符串，从前向后写和从后向前写都是一样的。例如”dad”,”racecar”,”1001”.

使用栈，可以轻松判断一个字符串是否为回文。我们将字符串每一个字符依次压入站，在入栈就保存了一个相反的字符串。我们只需要判断两个字符串是否相等即可。
