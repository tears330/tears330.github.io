JavaScript对象的深度克隆

发表于 2016-02-04 | 分类于 JavaScript |

JavaScript中一切实例都是对象，但是对象之间有所不同，分为基本类型和引用类型。基本类型就是常说的5中String,Number,Boolean,Undefined,Null。而引用类型往往指的是Object，Array，Function等等。
