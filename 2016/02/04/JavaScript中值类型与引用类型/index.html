JavaScript中值类型与引用类型

发表于 2016-02-04 | 分类于 JavaScript |

在JavaScript中变量有两种不同类型的值：值类型和引用类型值。基本类型值是指简单的数据段，而引用类型值值那些可能由多个值构成的对象。

JavaScript值类型和引用类型有哪些

  1. 值类型： Number,Boolean,Null,Undefined，String.
  2. 引用类型： Object,Array,Function…

值类型与引用类型的区别

  1. 值类型：值类型的值在内存中的存放内容即是实际值。即可以操作保存在变量中的实际值。

  2. 引用类型：引用类型的值是保存在内存中的对象。而JavaScript不允许直接访问内存位置。当复制保存对象的某个变量，操作的是对象的引用。即引用类型的值是按引用访问的。但在为对象添加属性时，操作的是实际对象。
